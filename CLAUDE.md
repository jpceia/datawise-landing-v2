# Datawise Landing Page v2

## Project Overview
Multi-language marketing website for Datawise, an AI/Data Science consulting startup.

## Tech Stack
- **Framework:** Next.js 14.2.x + React 18 + TypeScript
- **Styling:** Tailwind CSS 3.4.1
- **i18n:** next-intl (pt, en)
- **CMS:** Sanity (blog posts)
- **Animation:** Framer Motion
- **Analytics:** Google Analytics, GTM, Vercel Analytics
- **Lead capture:** HubSpot forms + Calendly

## Key Commands
```bash
npm run dev      # Development server
npm run build    # Production build
npm start        # Start production
npm run lint     # ESLint
```

## Environment Variables
