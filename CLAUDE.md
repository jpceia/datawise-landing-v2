# Datawise Landing Page v2

## Project Overview
Multi-language marketing website for Datawise, an AI/Data Science consulting startup.

## Tech Stack
- **Framework:** Next.js 14.2.30 + React 18 + TypeScript
- **Styling:** Tailwind CSS 3.4.1
- **i18n:** next-intl (pt, en, fr, es)
- **CMS:** Sanity (blog posts)
- **3D:** Three.js + React Three Fiber (Globe)
- **Animation:** Framer Motion, GSAP
- **Analytics:** Google Analytics, GTM, Vercel Analytics

## Key Commands
```bash
npm run dev      # Development server
npm run build    # Production build
npm start        # Start production
npm run lint     # ESLint
```

## Environment Variables
```
NEXT_PUBLIC_GA_ID
NEXT_PUBLIC_CALENDLY_URL
NEXT_PUBLIC_SANITY_PROJECT_ID
NEXT_PUBLIC_SANITY_DATASET
NEXT_PUBLIC_SANITY_API_VERSION
```
